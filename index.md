# Privacy Policy for MinimalNav

**Last updated:** 17 April 2026

MinimalNav is a Connect IQ application for Garmin devices that displays turn-by-turn navigation information in a simplified form. This policy describes how the application handles data.

## Data Collection

MinimalNav does not collect, store, or transmit any personal data.

The application reads navigation data (distance to the next waypoint, bearing, current heading, current speed) that is provided locally by the Garmin device while an active course is being navigated. This data is used exclusively to render the on-screen display in real time and is not stored, logged, or sent to any external service.

## Data Sharing

MinimalNav does not share any data with the developer or with third parties. The application has no network connectivity and makes no outbound connections.

## Permissions

The application requests the following Garmin Connect IQ permissions:

- **Positioning** — required to access the device's current heading and navigation state while a course is active. Location data never leaves the device.
